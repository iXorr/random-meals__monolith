## DESCRIPTION

This is an experimental project (simple landing website) created as part of a small team work.

We (I and my classmates) used the free API to fetch meal data and experimented with integrating an automatic translation module from npm. Interestingly, the translation results were often inaccurate (I would say - ridiculous), but in some cases, they produced unexpected and creative outputs that made the project even more unique. And just fun.

As a result, the project evolved into a playful and dynamic “food mixing” website with unpredictable content behavior.
